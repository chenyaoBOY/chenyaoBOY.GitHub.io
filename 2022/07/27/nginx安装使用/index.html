M1 安装NG

  • 首先MAC安装了brew
  • brew install nginx 安装NG
  • brew services start nginx 启动NG 访问 localhost:8080
  • NG配置 /opt/homebrew/etc/nginx/nginx.conf
  • 默认安装目录 /opt/homebrew/opt/nginx/

安装成功的提示：

基本命令操作

  • 启动 ./nginx
  • 关闭 ./nginx -s stop
  • 重启 ./nginx -s reload
  • nginx -V（大写） 可查看日志文件目录 配置文件目录等
